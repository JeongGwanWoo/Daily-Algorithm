# [BOJ] 1916 - 최소비용 구하기

### 문제 정보
* **문제 링크**: https://www.acmicpc.net/problem/1916
* **분류**: 그래프 이론, 다익스트라 (Dijkstra)
* **난이도**: Gold 5

### 학습 목적
* 가중치가 있는 방향 그래프에서 단일 출발점 최단 경로를 찾는 다익스트라 알고리즘의 이해
* 우선순위 큐(`PriorityQueue`)를 활용한 탐색 시간 최적화 (O(E log V)) 구현
* `Comparable` 인터페이스를 활용한 커스텀 객체(`State`)의 정렬 기준(비용 오름차순) 설정 방식 숙달
