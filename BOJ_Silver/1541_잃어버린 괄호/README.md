# [BOJ] 1541 - 잃어버린 괄호

### 문제 정보

* **문제 링크**: [https://www.acmicpc.net/problem/1541](https://www.acmicpc.net/problem/1541)
* **분류**: 그리디(Greedy), 수학, 문자열 파싱
* **난이도**: Silver 2

### 학습 목적

* 최솟값을 만들기 위한 수식의 괄호 배치 원리(그리디 알고리즘) 이해
* `split()` 메서드 및 정규표현식을 활용한 문자열 분리 숙달

### 코드 성능

* **메모리**: 14,188 KB
* **시간**: 100 ms
* **언어**: Java 8

### 핵심 설계 및 로직

* **그리디 전략**: 뺄셈(`-`) 기호를 기준으로 항을 묶어 합산한 뒤, 첫 번째 항에서 나머지 합산된 항들을 모두 빼야 최솟값이 도출됨.
* **파싱**:
1. `-`를 기준으로 문자열 분리.
2. 분리된 각 그룹 내에서 `+`를 기준으로 다시 숫자를 분리하여 합산.


* **연산**: 첫 번째 그룹은 결과값에 더하고, 이후 그룹부터는 결과값에서 차감.
