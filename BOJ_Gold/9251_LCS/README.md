# [BOJ] 9251 - LCS

### 문제 정보

* **문제 링크**: [https://www.acmicpc.net/problem/9251](https://www.acmicpc.net/problem/9251)
* **분류**: 다이나믹 프로그래밍 (DP)
* **난이도**: Gold 5

### 학습 목적

* 두 수열 사이의 최장 공통 부분 수열(Longest Common Subsequence)을 구하는 전형적인 DP 알고리즘 이해
* 2차원 배열을 활용한 상태 전이 및 점화식 도출 능력 숙달

### 코드 성능

* **메모리**: 18,480 KB (입력 문자열 길이에 비례)
* **시간**: 140 ms (Java 8 기준)
* **언어**: Java 8

### 핵심 설계 및 로직

* **DP 테이블 정의**: `dp[i][j]`는 첫 번째 문자열의 `i`번째 문자까지와 두 번째 문자열의 `j`번째 문자까지 고려했을 때의 LCS 길이임.
* **상태 전이**:
* 두 문자가 일치할 경우: `dp[i][j] = dp[i-1][j-1] + 1` (이전 단계 최장 길이에 +1)
* 두 문자가 불일치할 경우: `dp[i][j] = max(dp[i-1][j], dp[i][j-1])` (이전까지의 최적값 유지)


* **결과 도출**: 최종적으로 `dp[s1.length()][s2.length()]`가 정답이 되며, 코드 내 `max` 변수를 통해 실시간 업데이트함.
